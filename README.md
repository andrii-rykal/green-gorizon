# Green Horizon Ireland

**Green Horizon Ireland** is a static web project showcasing a content-driven presentation of modern farming and agriculture in Ireland.

## Description

The site is built as a multi-page static website with Home, About Us, Contact, Privacy, and Terms sections. This project is intended for a portfolio and resume, demonstrating HTML, CSS, JavaScript, responsive design, and simple front-end interactivity.

## Project Structure

- `index.html` — main landing page with agriculture content and key sections
- `about/index.html` — About Us page with mission and overview
- `contact/index.html` — Contact page with feedback form
- `privacy/index.html` — Privacy policy page
- `terms/index.html` — Terms of use page
- `bmodel/index.html` — business model page
- `css/style.css` — site styles
- `js/index.js` — JavaScript for interactivity
- `img/` — image assets and logo
- `fonts/` — font files

## Technologies

- HTML5
- CSS3
- JavaScript

## Key Features

- multi-page static website structure
- responsive layout for desktop and mobile views
- navigation menu and call-to-action buttons
- content sections focused on Irish agriculture and farming
- contact form with success message handling
- cookie consent overlay
- FAQ accordion interaction

## Deployment

This project is ready to be deployed using GitHub Pages.

- If the repository is named `username.github.io`, deploy from the `main` branch.
- If using a project site, deploy from the `gh-pages` branch or the `docs/` folder on the `main` branch.

## Getting Started

1. Open `index.html` in your web browser.
2. For local testing, optionally serve the project folder with a simple HTTP server or use direct file access.

> This project is suitable for a portfolio or resume as a demonstration of front-end development and web design skills.
